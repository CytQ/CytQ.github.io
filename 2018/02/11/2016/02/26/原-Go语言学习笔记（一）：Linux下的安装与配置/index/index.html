Golang在Linux下的安装与配置

下载

https://golang.org/dl/ 进入这个网址，选择对应的版本进行下载，需要翻墙。

安装

默认是建议安装到/usr/local

sudo tar <span class="hljs-attribute">-C</span> /usr/<span class="hljs-built_in">local</span> <span class="hljs-attribute">-xzf</span> go1<span class="hljs-number">.6</span><span class="hljs-built_in">.</span>linux<span class="hljs-attribute">-amd64</span><span class="hljs-built_in">.</span>tar<span class="hljs-built_in">.</span>gz`</pre>

## 验证

进入/usr/local/go

<pre class="prettyprint">`bin/go <span class="hljs-built_in">version</span> 查看版本号是否对应`</pre>

## 设置环境变量

需要设置4个环境变量: GOROOT, GOPATH, GOBIN, PATH 

设置到profile文件中(~/.bash_profile或者 /etc/profile) 

我这里选择的是/etc/profile

<pre class="prettyprint">`<span class="hljs-comment">#Go environment</span>
<span class="hljs-keyword">export</span> GOROOT=/usr/local/go
<span class="hljs-keyword">export</span> GOPATH=~/golib:~/goproject
<span class="hljs-keyword">export</span> GOBIN=~/gobin
<span class="hljs-keyword">export</span> PATH=<span class="hljs-variable">$PATH</span>:<span class="hljs-variable">$GOROOT</span>/bin:<span class="hljs-variable">$GOBIN</span>

添加完成之后，使用 source ./profile 让信息立刻生效

验证

在任意目录下输入 go version查看版本号

下载beego

go get github.com/astaxie/beego
