Python学习笔记(一)　高阶函数，偏函数，闭包

  • Python
  • Study Note

Python中高阶函数可接受其他的函数作为参数。

常用高阶函数

map(function, list)

让list的每一个元素依次调用function函数，并获取返回值存入一个新的list中。

<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">format_name</span><span class="hljs-params">(s)</span>:</span>
    <span class="hljs-keyword">return</span> s.capitalize() <span class="hljs-comment">#将首字母大写，其余小写</span>

<span class="hljs-keyword">print</span> map(format_name, [<span class="hljs-string">'adam'</span>, <span class="hljs-string">'LISA'</span>, <span class="hljs-string">'barT'</span>])`</pre>

结果为: [‘Adam’, ‘Lisa’, ‘Bart’]

## reduce(f, list, optional)

函数f必须包含两个参数，optional可选，如果存在则表示初值为optional 

reduce()对list的每个元素反复调用函数f，并返回最终结果值。

<pre class="prettyprint">`<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">f</span><span class="hljs-params">(x, y)</span>:</span>
    <span class="hljs-keyword">return</span> x + y

reduce(f, [<span class="hljs-number">1</span>, <span class="hljs-number">3</span>, <span class="hljs-number">5</span>, <span class="hljs-number">7</span>, <span class="hljs-number">9</span>])

<span class="hljs-comment">#先计算头两个元素：f(1, 3)，结果为4；</span>
<span class="hljs-comment">#再把结果和第3个元素计算：f(4, 5)，结果为9；</span>
<span class="hljs-comment">#再把结果和第4个元素计算：f(9, 7)，结果为16；</span>
<span class="hljs-comment">#再把结果和第5个元素计算：f(16, 9)，结果为25；</span>
<span class="hljs-comment">#由于没有更多的元素了，计算结束，返回结果25。</span>`</pre>

## filter(f, list)

函数 f 的作用是对每个元素进行判断，返回 True或 False，filter()根据判断结果自动过滤掉不符合条件的元素，返回由符合条件元素组成的新list。

<pre class="prettyprint">`<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">is_odd</span><span class="hljs-params">(x)</span>:</span>
    <span class="hljs-keyword">return</span> x % <span class="hljs-number">2</span> == <span class="hljs-number">1</span>

filter(is_odd, [<span class="hljs-number">1</span>, <span class="hljs-number">4</span>, <span class="hljs-number">6</span>, <span class="hljs-number">7</span>, <span class="hljs-number">9</span>, <span class="hljs-number">12</span>, <span class="hljs-number">17</span>])

<span class="hljs-comment">#结果　[1, 7, 9, 17]</span>`</pre>

## 自定义排序函数 sorted(list, optional)

<pre class="prettyprint">`<span class="hljs-tag">sorted</span>(<span class="hljs-attr_selector">[36, 5, 12, 9, 21]</span>)

# <span class="hljs-attr_selector">[5, 9, 12, 21, 36]</span>`</pre>

sorted()也是一个高阶函数，它可以接收一个比较函数来实现自定义排序，比较函数的定义是，传入两个待比较的元素 x, y，如果 x 应该排在 y 的前面，返回 -1，如果 x 应该排在 y 的后面，返回 1。如果 x 和 y 相等，返回 0。

如果我们要实现倒序排序，只需要编写一个reversed_cmp函数：

<pre class="prettyprint">`<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">reversed_cmp</span><span class="hljs-params">(x, y)</span>:</span>
    <span class="hljs-keyword">if</span> x &gt; y:
        <span class="hljs-keyword">return</span> -<span class="hljs-number">1</span>
    <span class="hljs-keyword">if</span> x &lt; y:
        <span class="hljs-keyword">return</span> <span class="hljs-number">1</span>
    <span class="hljs-keyword">return</span> <span class="hljs-number">0</span>

sorted([<span class="hljs-number">36</span>, <span class="hljs-number">5</span>, <span class="hljs-number">12</span>, <span class="hljs-number">9</span>, <span class="hljs-number">21</span>], reversed_cmp)

<span class="hljs-comment">#[36, 21, 12, 9, 5]</span>`</pre>

sorted()也可以对字符串进行排序，字符串默认按照ASCII大小来比较：

<pre class="prettyprint">`sorted([<span class="hljs-string">'bob'</span>, <span class="hljs-string">'about'</span>, <span class="hljs-string">'Zoo'</span>, <span class="hljs-string">'Credit'</span>])
<span class="hljs-array"># </span>[<span class="hljs-string">'Credit'</span>, <span class="hljs-string">'Zoo'</span>, <span class="hljs-string">'about'</span>, <span class="hljs-string">'bob'</span>]`</pre>

对字符串排序时，有时候忽略大小写排序更符合习惯。 

请利用sorted()高阶函数，实现忽略大小写排序的算法。

<pre class="prettyprint">`<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">cmp_ignore_case</span><span class="hljs-params">(s1, s2)</span>:</span>

<span class="hljs-keyword">print</span> sorted([<span class="hljs-string">'bob'</span>, <span class="hljs-string">'about'</span>, <span class="hljs-string">'Zoo'</span>, <span class="hljs-string">'Credit'</span>], cmp_ignore_case) 

<span class="hljs-comment"># ['about', 'bob', 'Credit', 'Zoo']</span>`</pre>

## 返回函数

请编写一个函数calc_prod(lst)，它接收一个list，返回一个函数，返回函数可以计算参数的乘积。 

内部编写的函数可使用外部的函数变量

<pre class="prettyprint">`<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">calc_prod</span><span class="hljs-params">(lst)</span>:</span>

print f()打印的是函数运行的结果，也就是返回值。

## 闭包

在函数内部定义的函数和外部定义的函数是一样的，只是他们无法被外部访问。

<pre class="prettyprint">`<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">calc_sum</span><span class="hljs-params">(lst)</span>:</span>
    <span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">lazy_sum</span><span class="hljs-params">()</span>:</span>
        <span class="hljs-keyword">return</span> sum(lst)
    <span class="hljs-keyword">return</span> lazy_sum`</pre>

这种内层函数引用了外层函数的变量（参数也算变量），然后返回内层函数的情况，称为闭包（Closure）。 

闭包的特点是返回的函数还引用了外层函数的局部变量，所以，要正确使用闭包，就要确保引用的局部变量在函数返回后不能变。返回函数不要引用任何循环变量，或者后续会发生变化的变量。

## 匿名函数：关键字 lambda

高阶函数可以接收函数做参数，有时候我们不需要显式地定义函数，直接传入匿名函数更方便。

关键字lambda 表示匿名函数，冒号前面的 x 表示函数参数。

匿名函数有个限制，就是只能有一个表达式，不写return，返回值就是该表达式的结果。

<pre class="prettyprint">`map(<span class="hljs-keyword">lambda</span> x: x * x, [<span class="hljs-number">1</span>, <span class="hljs-number">2</span>, <span class="hljs-number">3</span>, <span class="hljs-number">4</span>, <span class="hljs-number">5</span>, <span class="hljs-number">6</span>, <span class="hljs-number">7</span>, <span class="hljs-number">8</span>, <span class="hljs-number">9</span>])
<span class="hljs-comment"># [1, 4, 9, 16, 25, 36, 49, 64, 81]</span>
<span class="hljs-comment"># 匿名函数 lambda x: x * x 实际上就是：</span>
<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">f</span><span class="hljs-params">(x)</span>:</span>
    <span class="hljs-keyword">return</span> x * x`</pre>

<pre class="prettyprint">`<span class="hljs-comment">#利用匿名函数简化以下代码：</span>
<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">is_not_empty</span><span class="hljs-params">(s)</span>:</span>
    <span class="hljs-keyword">return</span> s <span class="hljs-keyword">and</span> len(s.strip()) &gt; <span class="hljs-number">0</span>

<span class="hljs-keyword">print</span> filter(
    <span class="hljs-keyword">lambda</span> s: s <span class="hljs-keyword">and</span> len(s.strip())&gt;<span class="hljs-number">0</span>,
    [<span class="hljs-string">'test'</span>, <span class="hljs-keyword">None</span>, <span class="hljs-string">''</span>, <span class="hljs-string">'str'</span>, <span class="hljs-string">'  '</span>, <span class="hljs-string">'END'</span>])`</pre>

## 偏函数

当一个函数有很多参数时，调用者就需要提供多个参数。如果减少参数个数，就可以简化调用者的负担。

例如，int(str, option)函数可以把字符串转换为整数，第二个参数option可选，如果设置了的话就可以设置函数按多少进制转换，默认的函数是十进制转换。

<pre class="prettyprint">`&gt;&gt;&gt; <span class="hljs-keyword">int</span>(<span class="hljs-string">'12345'</span>)
<span class="hljs-number">12345</span>
&gt;&gt;&gt; <span class="hljs-keyword">int</span>(<span class="hljs-string">'12345'</span>, <span class="hljs-keyword">base</span>=<span class="hljs-number">8</span>)
<span class="hljs-number">5349</span>
&gt;&gt;&gt; <span class="hljs-keyword">int</span>(<span class="hljs-string">'12345'</span>, <span class="hljs-number">16</span>)
<span class="hljs-number">74565</span>`</pre>

假设要转换大量的二进制字符串，每次都传入int(x, base=2)非常麻烦，于是可以定义一个int2()的函数，默认把base=2传进去。

<pre class="prettyprint">`<span class="hljs-function"><span class="hljs-keyword">def</span> <span class="hljs-title">int2</span><span class="hljs-params">(x, base=<span class="hljs-number">2</span>)</span>:</span>
    <span class="hljs-keyword">return</span> int(x, base)`</pre>

而从python来说，提供了一种定义好的方法，帮助我们创建一个偏函数的，不需要我们自己定义一个新函数，可以直接使用下面的代码创建一个新的函数：

<pre class="prettyprint">`import functools
int2 = functools.<span class="hljs-keyword">partial</span>(<span class="hljs-keyword">int</span>, <span class="hljs-keyword">base</span>=<span class="hljs-number">2</span>)`</pre>

在sorted这个高阶函数中传入自定义排序函数可以实现忽略大小写排序。可用functools.partial把这个复杂调用变成一个简单的函数：

<pre class="prettyprint">`<span class="hljs-keyword">import</span> functools

sorted_ignore_case = functools.partial(sorted, 
cmp=<span class="hljs-keyword">lambda</span> s1, s2: cmp(s1.upper(), s2.upper()))

<span class="hljs-keyword">print</span> sorted_ignore_case([<span class="hljs-string">'bob'</span>, <span class="hljs-string">'about'</span>, <span class="hljs-string">'Zoo'</span>, <span class="hljs-string">'Credit'</span>])
